Manual Configuration & REST API

If Apple Shortcuts don't download, you can design them manually inside the Apple Shortcuts App:

Shortcut A: "Send to PC" (Share Sheet)

  1. Open Shortcuts App → tap +
  2. Open shortcut settings ➜ turn on "Show in Share Sheet" ➜ set input type to Any
  3. Add action: "If" ➜ choose Shortcut Input ➜ click key → set "File Extension" to "has any value"
  4. Under Then (Files/Images path):
    • Add: "Get Contents of URL"
    • URL: http://...:3479/api/send
    • Method: POST → Request Body: File → File: Shortcut Input
  5. Under Else (Text/URL path):
    • Add: "Get Contents of URL"
    • URL: http://...:3479/api/send
    • Method: POST → Request Body: Form → Text Field: Key: content, Value: Shortcut Input

Shortcut B: "Send Clipboard" (Home Widget)

  1. Open Shortcuts App → tap +
  2. Add: "Get Clipboard"
  3. Add: "If" ➜ choose Clipboard ➜ set "File Extension" to "has any value"
  4. Under Then (Files/Images path):
    • Add: "Get Contents of URL"
    • URL: http://...:3479/api/send
    • Method: POST → Request Body: File → File: Clipboard
  5. Under Else (Text/URL path):
    • Add: "Get Contents of URL"
    • URL: http://...:3479/api/send
    • Method: POST → Request Body: Form → Text Field: Key: content, Value: Clipboard

Shortcut C: "Receive from PC" (Unified, Handles Text & Files)

  1. Open Shortcuts App → tap +
  2. Add: "Get Contents of URL" ➜ URL: http://...:3479/api/clipboard ➜ Method: GET
  3. Add: "Get Dictionary from Input" ➜ set input to Contents of URL
  4. Add: "Get Dictionary Value" ➜ set key to success
  5. Add: "If" ➜ set condition to is true:
    • Add: "Get Dictionary Value" ➜ set key to type and dictionary to the top-level Dictionary
    • Add: "If" ➜ set condition to is text:
      • Add: "Get Dictionary Value" ➜ set key to text
      • Add: "Copy to Clipboard" ➜ set target to Dictionary Value
      • Add: "Get Dictionary Value" ➜ set key to id
      • Add: "Get Contents of URL" ➜ URL: http://...:3479/api/pending/<id>/ack ➜ Method: POST
      • Add: "Show Notification" ➜ Message: Text copied to clipboard
    • Under Otherwise (File path):
      • Add: "Choose from Menu" ➜ Prompt: Download File? ➜ Options: Download, Cancel
      • Under Download:
        • Add: "Get Dictionary Value" ➜ set key to url
        • Add: "Get Contents of URL" ➜ set target to the Dictionary Value (downloads file)
        • Add: "Get Dictionary Value" ➜ set key to mimeType
        • Add: "If" ➜ set condition to starts with image/ or video/:
          • Add: "Save to Photo Album" ➜ input: downloaded Contents of URL
        • Otherwise:
          • Add: "Save File" ➜ input: downloaded Contents of URL
        • Add: "Get Dictionary Value" ➜ set key to id
        • Add: "Get Contents of URL" ➜ URL: http://...:3479/api/pending/<id>/ack ➜ Method: POST
        • Add: "Show Notification" ➜ Message: File downloaded successfully
  6. Under outer Otherwise (Empty state):
    • Add: "Show Notification" ➜ Message: Clipboard is empty
