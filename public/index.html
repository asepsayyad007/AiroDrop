Manual Shortcut Setup

iOS 15+ has removed support for direct third-party shortcut downloads. Please configure the shortcuts manually in the Shortcuts app:

Shortcut 1: "Send to PC" (Share Sheet)

  1. Open Shortcuts app → tap +
  2. Tap ⓘ ➜ turn on "Show in Share Sheet" ➜ set input to accept Any
  3. Add action: "If" ➜ choose Shortcut Input ➜ tap the variable inside "If" and change it to "File Extension" ➜ set condition to "has any value"
  4. Inside "Then" (File/Media branch):
    • Add: "Get Contents of URL"
    • URL: http://...:3478/api/send
    • Method: POST → Request Body: File → File: Shortcut Input
  5. Inside "Else" (Text branch):
    • Add: "Get Contents of URL"
    • URL: http://...:3478/api/send
    • Method: POST → Request Body: Form → Add Text Field: Key: content, Value: Shortcut Input

Shortcut 2: "Send Clipboard" (Home Screen Widget)

  1. Open Shortcuts app → tap +
  2. Add action: "Get Clipboard"
  3. Add action: "If" ➜ choose Clipboard ➜ tap the variable inside "If" and change it to "File Extension" ➜ set condition to "has any value"
  4. Inside "Then" (File/Media branch):
    • Add: "Get Contents of URL"
    • URL: http://...:3478/api/send
    • Method: POST → Request Body: File → File: Clipboard
  5. Inside "Else" (Text/Links branch):
    • Add: "Get Contents of URL"
    • URL: http://...:3478/api/send
    • Method: POST → Request Body: Form → Add Text Field: Key: content, Value: Clipboard
