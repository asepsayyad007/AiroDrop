Manual Setup & REST API

💡 Easy Tip: Use Tab 1 for 1-tap download links! If you prefer creating shortcuts manually in the Apple Shortcuts App or writing custom code, follow these simple steps:

Pass Your Secret Key (If Security is On)

If you created a Secret Key above, add it to your HTTP request:
• Header: Key: X-AiroDrop-Token | Value: <Your Secret Key>
• Or URL: http://<PC-IP>:3479/api/send?token=<Your Secret Key>

1. "Send to PC" Shortcut (Share Sheet)

  1. Open Shortcuts App on iPhone → tap +
  2. Tap shortcut settings (top) ➜ turn on "Show in Share Sheet"
  3. Add action: "If" ➜ set input to Shortcut Input ➜ choose attribute "File Extension" ➜ condition: "has any value"
  4. Under Then (for Photos & Files):
    • Add action: "Get Contents of URL"
    • URL: http://...:3479/api/send
    • Method: POST → Body: File → File: Shortcut Input
    • Header (if secret set): Key X-AiroDrop-Token, Value: Your Secret Key
  5. Under Else (for Copied Text & Web Links):
    • Add action: "Get Contents of URL"
    • URL: http://...:3479/api/send
    • Method: POST → Body: Form → Key content, Value: Shortcut Input
    • Header (if secret set): Key X-AiroDrop-Token, Value: Your Secret Key

2. "Send Clipboard" Shortcut (Home Widget)

  1. Open Shortcuts App → tap +
  2. Add action: "Get Clipboard"
  3. Add action: "If" ➜ set input to Clipboard ➜ attribute "File Extension" ➜ condition: "has any value"
  4. Under Then (Files/Images):
    • Add: "Get Contents of URL" ➜ URL: http://...:3479/api/send
    • Method: POST → Body: File → File: Clipboard
    • Header (if secret set): Key X-AiroDrop-Token, Value: Your Secret Key
  5. Under Else (Text):
    • Add: "Get Contents of URL" ➜ URL: http://...:3479/api/send
    • Method: POST → Body: Form → Key content, Value: Clipboard
    • Header (if secret set): Key X-AiroDrop-Token, Value: Your Secret Key

3. "Get From PC" Shortcut (Receive Text & Files)

  1. Open Shortcuts App → tap +
  2. Add action: "Get Contents of URL" ➜ URL: http://...:3479/api/clipboard ➜ Method: GET (Header if secret set: Key X-AiroDrop-Token, Value: Your Secret Key)
  3. Add action: "Get Dictionary from Input" ➜ set input to Contents of URL
  4. Add action: "Get Dictionary Value" ➜ set key to success
  5. Add action: "If" ➜ set condition to is true:
    • Add: "Get Dictionary Value" ➜ set key to type
    • Add: "If" ➜ set condition to is text:
      • Add: "Get Dictionary Value" ➜ key text → "Copy to Clipboard"
      • Add: "Get Contents of URL" ➜ URL: http://...:3479/api/pending/<id>/ack ➜ Method: POST
      • Add: "Show Notification" ➜ "Text copied to clipboard"
    • Under Otherwise (If File):
      • Add: "Choose from Menu" ➜ Prompt: Download File? ➜ Options: Download / Cancel
      • Under Download:
        • Add: "Get Dictionary Value" ➜ key url → "Get Contents of URL"
        • Add: "Get Dictionary Value" ➜ key mimeType
        • Add: "If" ➜ starts with image/ or video/ → "Save to Photo Album"; Else → "Save File"
        • Add: "Get Contents of URL" ➜ URL: http://...:3479/api/pending/<id>/ack ➜ Method: POST
        • Add: "Show Notification" ➜ "File downloaded"

⚡ REST API Endpoint Reference

POST /api/send
Send text or files to PC. For form text: content=hello. For raw binary file: send file stream in body. Pass X-AiroDrop-Token header if secret set.
GET /api/clipboard
Fetch current active text or pending transfer item queued from PC to mobile. Pass X-AiroDrop-Token header if secret set.
POST /api/pending/:id/ack
Acknowledge receipt of a queued item from PC to clear it from queue. Pass X-AiroDrop-Token header if secret set.
