Manual Configuration & REST API

If Apple Shortcuts don't download, you can design them manually inside the Apple Shortcuts App:

Shortcut A: "Send to PC" (Share Sheet)

  1. Open Shortcuts App → tap +
  2. Open shortcut settings ➜ turn on "Show in Share Sheet" ➜ set input type to Any
  3. Add action: "If" ➜ choose Shortcut Input ➜ click key → set "File Extension" to "has any value"
  4. Under Then (Files/Images path):
    • Add: "Get Contents of URL"
    • URL: http://...:3478/api/send
    • Method: POST → Request Body: File → File: Shortcut Input
  5. Under Else (Text/URL path):
    • Add: "Get Contents of URL"
    • URL: http://...:3478/api/send
    • Method: POST → Request Body: Form → Text Field: Key: content, Value: Shortcut Input

Shortcut B: "Send Clipboard" (Home Widget)

  1. Open Shortcuts App → tap +
  2. Add: "Get Clipboard"
  3. Add: "If" ➜ choose Clipboard ➜ set "File Extension" to "has any value"
  4. Under Then (Files/Images path):
    • Add: "Get Contents of URL"
    • URL: http://...:3478/api/send
    • Method: POST → Request Body: File → File: Clipboard
  5. Under Else (Text/URL path):
    • Add: "Get Contents of URL"
    • URL: http://...:3478/api/send
    • Method: POST → Request Body: Form → Text Field: Key: content, Value: Clipboard
